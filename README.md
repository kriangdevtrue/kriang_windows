## Kriang Windows App

https://kriang-windows.herokuapp.com/

Application ช่วยงานช่างอลูมิเนียม ในการคำนวณการตัดชิ้นงาน เส้นอลูมิเนียม กระจก และจำนวนอะไหล่ต่างๆที่ต้องใช้ สำหรับงานประตูและหน้าต่างอลูมิเนียม เปรียบเสมือนคู่มือช่าง ช่วยลดความผิดพลาดในการทำงาน ได้ผลงานที่มีคุณภาพ

> Features
1. เพิ่มประเภทและแบบบานประตู หน้าต่างได้
2. ใส่สูตรการคำนวณตัดชิ้นงาน และระบุจำนวนที่ต้องใช้
3. รองรับการระบุสูตรสำหรับอลูมิเนียม และกระจก
4. ฟังก์ชั่นช่วยตรวจสอบสูตร และแสดงผลลัพธ์ที่ได้
